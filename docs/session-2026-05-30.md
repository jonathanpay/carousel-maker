## About the tool

**Carousel Maker** is a browser-based tool for designing and exporting social media carousel slides. No build step — open an HTML file in a browser.

### Template families
| Key | Label | Style |
|---|---|---|
| `jp-editorial` | JP — Editorial | Refined navy + gold |
| `jp-magazine` | JP — Magazine | Luxe oversized italic |
| `hea-clean` | HEA — Clean | Bright purple, polished |
| `hea-shapes` | HEA — Shapes | Dark + geometric blocks |

### Layouts (10)
Cover · Big Stat · Numbered Tip · Checklist · Pull Quote · Compare · Case Study · Image + Caption · Author · CTA

### Key files
- `Carousel Maker.html` — main entry point
- `Carousel Maker-standalone.html` — bundled, self-contained version
- `Carousel Maker-print.html` — print/PDF layout
- `app.js` / `templates.js` — logic and render functions
- `app.css` / `templates.css` — styles

### Notable features
- `*word*` syntax in text fields renders in the theme accent colour
- Tweaks panel: font scale, padding scale, show/hide page numbers and handles
- PNG export via html2canvas (single slide or all)
- Template browser modal (4 families × 10 layouts)
